# 🚀 超简单部署指南 - 五分钟上线！

## 📋 方式一：GitHub Desktop（推荐！最简单！

### 第一步：下载安装 GitHub Desktop

1. 打开浏览器，访问：https://desktop.github.com/
2. 点击 "Download for Windows" 下载
3. 安装并启动 GitHub Desktop
4. 用您的 GitHub 账号登录

### 第二步：添加项目到 GitHub Desktop

1. 在 GitHub Desktop 中，点击 **File** → **Add Local Repository...**
2. 在弹出的窗口中，点击 **Choose...**
3. 选择您的项目文件夹：`c:\Users\xzy\Desktop\2\项目代码`
4. 点击 **Add Repository**

### 第三步：发布到 GitHub！

1. 现在 GitHub Desktop 应该会显示您的项目
2. 点击中间上方的 **"Publish repository"** 按钮
3. 填写信息：
   - **Name**: `gwy-recommendation-system`（或您喜欢的名字）
   - **Description**: `公务员考试岗位个性化推荐系统`
   - **Keep this code private?**: 保持 **不勾选**（Public，免费！
4. 点击 **"Publish Repository"** 按钮！
5. ✅ 等待发布完成！

### 第四步：去 Streamlit Cloud 部署！

现在 GitHub 仓库已经准备好了！

1. **打开浏览器**，访问：https://share.streamlit.io/
2. **用 GitHub 登录**
3. **点击 "New app"**（或 "Create app"）
4. **填写部署信息：
   - Repository: 选择刚才创建的仓库（如：您的用户名/gwy-recommendation-system）
   - Branch: 选择 `main`
   - Main file path: 填 `src/app.py` ⚠️ 重要！
5. **点击 "Deploy!"** 按钮！
6. **等待 2-5 分钟...**
7. **🎉 完成！**

您会获得一个链接，类似：https://gwy-recommendation-system.streamlit.app/
